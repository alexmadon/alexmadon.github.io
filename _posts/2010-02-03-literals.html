The syntax of a function literal in Scala (step 6 p69) Examples:
  1. Identity: 
    scala> val g = (i:Int) => i
g: (Int) => Int = <function>
  2. Is even literal: 
    scala> val g = (i:Int) => (i % 2) == 0
g: (Int) => Boolean = <function>
scala> g(1)
res8: Boolean = false
