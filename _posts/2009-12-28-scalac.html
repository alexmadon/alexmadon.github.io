package com.atpic.tutorial

object Out {
  def sayhello(who: String) = "Hello "+who
}
This a singleton object (see http://www.artima.com/scalazine/articles/steps.html then compile it: 
scalac tutorial.scala
Then you can call it: 
scala -cp .
