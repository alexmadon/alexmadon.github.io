0) install via USB the software (e.g using a win8 laptop)
1) Find the IP of your wifi router (modem)
root@madona:~# route -n
Kernel IP routing table
Destination Gateway Genmask Flags Metric Ref Use Iface
0.0.0.0 192.168.1.1 0.0.0.0 UG 0 0 0 wlan0
192.168.1.0 0.0.0.0 255.255.255.0 U 0 0 0 wlan0

2) connect to the web interface management of that IP:
http://192.168.1.1

3) go to the list of connected devices and get the IP of the printer

4) confirm this is a printer

root@madona:~# nmap 192.168.1.14
